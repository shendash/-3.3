Это тело аккордеона первого элемента. Оно отображается по умолчанию, пока плагин collapse не добавит соответствующие классы, которые мы используем для оформления каждого элемента. Эти классы управляют общим внешним видом, а также отображением и скрытием с помощью CSS-переходов. Вы можете изменить что-либо из этого с помощью пользовательского CSS или переопределив наши переменные по умолчанию. Также стоит отметить, что практически любой HTML-код может быть включен в .accordion-body, хотя переход ограничивает переполнение.

This is the third item's accordion body. It is hidden by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.
